# 🍣 寿司打 ＆ pop タイピング

ローマ字で日本語を打つ、ブラウザだけで動くタイピングゲームです。
サーバー・ビルド不要。`index.html` を開くだけで遊べます。

## 遊び方
1. モードを選ぶ（🍣 寿司打 / 💥 pop）
2. 難易度またはテーマ（カテゴリ）を選ぶ
3. スタート → 表示されたお題をローマ字で入力
   - 例: `寿司` → `sushi`、`し` は `shi` でも `si` でもOK
   - スマホは画面キーボードをタップ

## 主な機能
- **寿司打モード**: 流れるお題を時間内に打って食べる（お手軽/お勧め/高級コース、元を取れるか判定）
- **popモード**: 次々ポップするお題を弾く。コンボでハイスコア。見た目テーマ **💥ポップ / 🔫ガン / ⚔️刀** を選択可
- **難易度5段階（自動生成）**: 正しい読みの部品＋文型を実行時に組み合わせ、約11.9億通りの重複しないお題を生成
- **テーマ別カテゴリ**: 寿司ネタ / やさしい言葉 / **学年別漢字（教育漢字1,026字・全種類）** / ビジネス定型文 / 歴史の人物 / カタカナ語
- 効果音・BGM（WebAudioで合成／外部ファイル不要）、ハイスコア保存（localStorage）＋ランキング
- 漢字表示＋ふりがな、複数のローマ字入力パターン対応（促音・撥音・拗音・長音）

## ファイル構成
| ファイル | 役割 |
|---|---|
| `index.html` | 本体（UI・ゲームロジック・入力エンジン） |
| `wordgen.js` | 難易度別お題の自動生成エンジン |
| `kanji-grades.js` | 学年別漢字1,026字＋読みデータ |

3ファイルを同じフォルダに置けば動作します。

## GitHub Pages で公開する手順
1. GitHubで新しいリポジトリを作成（例: `typing-game`、Public）
2. このフォルダを push:
   ```bash
   git remote add origin https://github.com/<あなたのユーザー名>/typing-game.git
   git branch -M main
   git push -u origin main
   ```
3. GitHubのリポジトリ → **Settings** → **Pages** →
   **Build and deployment** の Source を「Deploy from a branch」、Branch を **main / (root)** に設定して Save
4. 数十秒後、`https://<あなたのユーザー名>.github.io/typing-game/` で公開されます

> Netlify / Cloudflare Pages / Vercel でも、このフォルダをドラッグ&ドロップするだけで公開できます（ビルド設定不要）。
